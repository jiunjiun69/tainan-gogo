6/20 台南 GO

目前先以 A/C 二選一收斂；A 市區穩健版最有機會，C 高鐵周邊保留省心備案。

首選方向

先用 A/C 收斂方向。

目前比較像在「市區穩健」和「高鐵周邊」之間選，市區版更適合把午餐、晚餐、宵夜和御守散步都接起來；高鐵周邊版則保留給想少移動的情況。

選一個方向

A/B/C 方案

A 目前最可能

市區穩健版

早餐各自處理，午餐或早午餐進市區，下午接御守和散步，晚餐海鮮，宵夜看牛肉湯、夜市或打茶趣。

  • 餐飲選項最多
  • 御守和小吃好接
  • 晚餐海鮮最順
B 海線備案

七股海線版

七股鹽山、海產街、鹽田或海線風景，適合有人想真的跑景點。

  • 需要車或包車
  • 6 月白天會熱
  • 比較有出遊感
C 第二候選

高鐵周邊版

高鐵台南站、三井、奇美或十鼓、打茶趣，少移動，回程最乾淨；想加台南小吃就把宵夜拉回市區。

  • 適合晚到早走
  • 不太進市區
  • 集合壓力最低

行程草案

三套時程草案

A

市區穩健版

目前最適合當主案，吃喝和景點都能臨場調整。

早上早餐各自處理；若連假人潮較多，可以提早出門再到市區會合
11:30午餐/早午餐：法國鬍子、國華街小吃，或先用牛肉湯/鱔魚意麵開場
14:00延平郡王祠，順便看御守和附近街區
15:20孔廟、府中街、國華街甜點飲料：豆花、冰淇淋、紅茶、黑糖奶
17:40晚餐：海鮮、魚湯、熱炒或生魚片；蝦蟹類先不要當唯一主軸
20:00宵夜可有可無：牛肉湯、花園/武聖夜市，或打茶趣晚茶收尾

關鍵：先決定晚餐海鮮的形式，再從午餐和宵夜補台南小吃。這版最適合想吃完整、也想去御守景點的人。

B

七股海線版

適合想跑景點，但交通和天氣條件要先確認。

10:30集合取車/叫車，先確認有人願意跑海線
11:30午餐：七股海產街、阿月虱目魚粥、黑琵食堂方向
13:10七股鹽山，6/20 剛好落在官方活動期間
15:00井仔腳瓦盤鹽田、青鯤鯓扇形鹽田，依天氣二選一
17:30回市區晚餐，或直接回高鐵周邊吃飯
20:00高鐵回程；還有精神再接打茶趣

關鍵：這版需要車或包車。若當天太熱或下雨，建議直接切回 A 或 C。

C

高鐵周邊版

第二候選，不追求市區感，主打集合容易和回程簡單。

早上早餐各自處理；不進市區時，集合壓力最低
12:00高鐵台南站集合，三井 Outlet 午餐或先喝飲料
13:30奇美博物館、十鼓仁糖文創園區，依大家興趣二選一
16:30回高鐵/三井周邊休息，補甜點或咖啡
17:30打茶趣，吃烤奶、圍爐煮茶，當晚餐或晚茶
20:00回高鐵台南站；若還想吃台南味，再加碼市區牛肉湯或夜市

關鍵：這版很舒服，但市區小吃和御守感會少很多。適合大家只想輕鬆見面、不要跑太多點。

餐飲選項

吃喝候選

A 版吃法

市區完整吃喝線

  • 早餐各自處理，11:00-12:00 之間集合比較實際。
  • 午餐可走法國鬍子、牛肉湯、鱔魚意麵或國華街小吃。
  • 晚餐主軸放海鮮、魚湯、熱炒或生魚片，蝦蟹類先確認接受度。
  • 宵夜可有可無，牛肉湯、夜市、打茶趣都能當收尾。
C 版吃法

高鐵周邊省心線

  • 高鐵集合後先用三井餐廳解決午餐，最不怕有人晚到。
  • 下午接奇美或十鼓，晚餐/晚茶用打茶趣收尾。
  • 若還想吃台南小吃，晚上再回市區補牛肉湯或夜市。
  • 優點是輕鬆；代價是國華街、御守、市區感會少很多。
A 版午餐/下午茶

法國鬍子

安平甜點、咖啡、簡餐方向，適合排成早午餐或下午茶。若要當正餐，先確認預約、菜單和當天供餐狀況。

定位：A 版可排在市區散步前後；C 版除非願意進安平，否則不硬塞。

參考
A 版晚餐

海鮮 / 魚湯 / 熱炒

晚餐最適合當 A 版主軸。可以用弘生魚片、山記魚仔店，或其他魚湯熱炒餐廳做候選。

狀態：先確認要生魚片、魚湯、熱炒，蝦蟹類不要當唯一主菜。

C 版收尾

打茶趣

烤奶、圍爐煮茶，靠近高鐵台南站。適合 C 版主軸，也能當 A 版最後晚茶收尾。

狀態：建議先問訂位；營業時間以店家公告為準。

參考
宵夜 / 早餐

牛肉湯

很適合當 A 版宵夜，或隔天早餐。若晚上還想吃台南味，牛肉湯比再塞一餐大餐更彈性。

候選：文章、阿村、永樂一帶牛肉湯；營業時間差異大，出發前再定店。

參考
午餐 / 晚餐

鱔魚意麵

台南感很強，適合放午餐或宵夜前的小吃段。國華街附近可看福昇小食、進福炒鱔魚等方向。

適合：A 版市區小吃線；怕排隊就當備案，不要卡死整段時程。

參考
國華街小吃

金得春捲 / 富盛號碗粿 / 阿松割包

國華街經典小吃組，適合多人分食。若午餐不想坐很久，可以拆成邊走邊吃。

適合：A 版午餐、下午補食；連假排隊要抓鬆。

參考
甜點飲料

修安豆花 / 蜷尾家

適合放在延平郡王祠、孔廟、國華街之間當休息點。天氣熱時，比硬排下一個景點更友善。

適合：A 版下午茶；排隊太長就換飲料或冰品。

散步小吃

炸雞洋行

國華街人氣炸雞，適合當下午或宵夜前的小點，不建議取代正式晚餐。

注意：國華攤常見資訊為中午到傍晚，售完與店休要再看公告。

注意：夜市營業日與攤位仍以當日公告為準。
