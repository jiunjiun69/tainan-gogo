# 6/20 台南 GO

這個 repo 是 2026-06-20 台南一日遊的輕量行程頁。目標是把既有想法整理成三個可選方向：市區、七股海線、高鐵周邊。

## 目前建議

建議先採用 **A. 市區穩健版**：

- 以台南市區為主，交通與天氣風險最低。
- 延平郡王祠適合排進下午市區動線。
- 晚餐可以從生魚片/海鮮、台南小吃、打茶趣收尾之間調整。
- 七股鹽山保留為 B 方案，因為 6/20 剛好落在官方活動期間，但需要車與抗熱意願。

## 建議下一步

1. 先讓大家選 A/B/C，不要一開始開放式問「想去哪」。
2. 選完方向後，從吃喝池挑主餐與 1-2 個備案。
3. 連假期間主餐先訂位，打茶趣若要去也先問位置。
4. 出發前 3-5 天再確認營業、天氣、交通與回程時間。
