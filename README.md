# 6/20 台南 GO

這個 repo 是 2026-06-20 台南一日遊的輕量行程頁。目標是把既有想法整理成可快速比較的方向，目前以 A 市區穩健版與 C 高鐵周邊版最有參考價值。

## 目前建議

建議先採用 **A. 市區穩健版**：

- 以台南市區為主，交通與天氣風險最低。
- 延平郡王祠適合排進下午市區動線。
- 午餐可從法國鬍子、牛肉湯、鱔魚意麵、國華街小吃之間挑。
- 晚餐可以從海鮮、魚湯、熱炒、生魚片之間調整。
- 宵夜可用牛肉湯、夜市或打茶趣收尾；也可以直接回程。
- 七股鹽山保留為 B 方案，因為 6/20 剛好落在官方活動期間，但需要車與抗熱意願。

## 建議下一步

1. 先讓大家在 A/C 之間收斂，B 保留給真的想跑海線時啟用。
