# 地點池與查證

最後查證：2026-06-13。餐廳與景點資訊可能變動，出發前 3-5 天請重新確認營業、訂位、票務與天氣。

## 已有方向

### 打茶趣

- 類型：烤奶、圍爐煮茶、晚茶/晚餐。
- 適合方案：A 收尾、C 主軸、B 回高鐵前備案。
- 地址：臺南市歸仁區智慧綠能大道二段500巷20號。
- 判斷：烤奶與圍爐煮茶適合作為晚茶收尾；地點靠近高鐵台南站，放最後一站合理。
- 注意：營業時間來源不完全一致，建議以店家公告或電話確認。
- 來源：[Fooday 店家頁](https://fooday.app/zh-TW/spots/YVB6jTH79GrTVi6xQ4Z23J/menu)、[愛食記文章](https://ifoodie.tw/post/66cd778feb74c46aa8a7ace2)

### 生魚片 / 海鮮

- 類型：餐飲方向，不是單一地點。
- 適合方案：A 午餐或晚餐、B 海線午餐。
- 判斷：生魚片與海鮮可以變成餐廳主題。
- 注意：若午餐吃魚湯/生魚片，晚餐建議換台菜、小吃或打茶趣，不要整天同一類。

### 七股鹽山

- 類型：海線景點。
- 適合方案：B 主景點。
- 地址：台南市七股區鹽埕里66號。
- 判斷：官方網站列「七股鹽山鹹轉海風祭活動」時間為 2026-05-23 到 2026-06-21，6/20 剛好落在期間內。
- 注意：需要車與防曬；若天氣太熱或下雨，建議切回 A/C。
- 來源：[七股鹽山官方網站](https://cigu.tybio.com.tw/)

### 延平郡王祠

- 類型：市區景點。
- 適合方案：A 下午。
- 地址：臺南市中西區開山路152號。
- 判斷：御守和市區散步可以一起安排，排進 A 版很自然。
- 目前資訊：台南旅遊網列開放時間 08:30-17:30，免門票。
- 來源：[台南旅遊網：延平郡王祠](https://www.twtainan.net/zh-tw/Attractions/Detail/792/)

## 吃喝候選

### 法國鬍子

- 類型：甜點、咖啡、簡餐、早午餐/下午茶方向。
- 適合方案：A 午餐或下午茶。
- 判斷：適合想坐下來吃、聊天、不要一直排小吃時使用；若要當正餐，先確認預約、菜單與當天供餐。
- 注意：位置在安平方向，不適合 C 版硬塞。
- 來源：[愛食記店家頁](https://ifoodie.tw/restaurant/5cf8e16c226139711a1ff47c-La-Belle-Maison-Cafe)

### 牛肉湯

- 類型：宵夜、早餐、台南小吃。
- 適合方案：A 宵夜、A 午餐備案、C 加碼市區宵夜。
- 判斷：如果不想把宵夜吃成大餐，牛肉湯是很好的台南味補點。
- 注意：各店營業時間差異很大，出發前再從文章、阿村、永樂一帶牛肉湯等候選中挑實際店家。
- 來源：[聯合新聞網整理](https://travel.udn.com/travel/story/7208/9529275)、[七分之二旅遊整理](https://www.twosevenths.com/service/restaurant-tainan-beefsoup/)

### 鱔魚意麵

- 類型：台南小吃、午餐、晚餐前小吃。
- 適合方案：A 午餐或散步小吃。
- 判斷：國華街附近有福昇小食等候選，適合想吃台南感但不想坐很久時安排。
- 注意：排隊與營業時間要確認，不建議把整段行程卡死在單一店家。
- 來源：[慢遊台南整理](https://tainanroam.com/fried-eel/)、[福昇小食參考](https://www.tony60533.com/fusheng-tainan/)

### 國華街小吃金三角

- 類型：金得春捲、富盛號碗粿、阿松割包。
- 適合方案：A 午餐或下午補食。
- 判斷：地點集中、適合多人分食，可以把午餐拆成小吃巡禮。
- 注意：連假排隊要抓鬆，且不要和晚餐海鮮吃太近。
- 來源：[Round Taiwan Round 國華街整理](https://www.rtaiwanr.com/zh-TW/blog/guohua-street-food)、[食尚玩家整理](https://supertaste.tvbs.com.tw/amp/pack/340915)

### 修安豆花 / 蜷尾家 / 飲料甜點

- 類型：甜點、冰品、飲料休息點。
- 適合方案：A 下午。
- 判斷：延平郡王祠、孔廟、國華街之間需要休息點，甜點飲料比再塞景點更有彈性。
- 來源：[台南旅遊網：修安扁擔豆花](https://www.twtainan.net/only-for-member/zh-tw/shop/consume/1087)、[蜷尾家 FonFood 店家頁](https://www.fonfood.com/store/934482)

### 炸雞洋行

- 類型：國華街散步小吃。
- 適合方案：A 下午或宵夜前小點。
- 判斷：適合當加碼，不適合取代正式晚餐。
- 注意：常見資訊為中午到傍晚且售完為止，出發前請再查公告。
- 來源：[WacowTainan 店家頁](https://www.wtainan.com.tw/web/storeInfo/174)、[食尚玩家資訊](https://supertaste.tvbs.com.tw/infocard/13047)

### 花園夜市 / 武聖夜市

- 類型：週六宵夜備案。
- 適合方案：A 晚餐後加碼。
- 判斷：6/20 是週六，若大家晚餐後還有體力，可用夜市補東山鴨頭、白糖粿等小吃。
- 注意：夜市營業與攤位以當日公告為準。
- 來源：[台南旅遊網：花園夜市](https://twtainan.net/en/attractions/detail/5573)、[輕旅行夜市整理](https://travel.yam.com/article/131495)

- 判斷：符合生魚片/魚湯方向，也靠近延平郡王祠。
- 注意：不同來源顯示營業時間不完全一致，出發前請再查地圖資訊或電話。
- 來源：[Taiwan Foodie](https://taiwanfoodie.org/restaurants/shan_ji_fish_shop/)、[Uber Eats](https://www.ubereats.com/tw-en/store/%E5%B1%B1%E8%A8%98%E9%AD%9A%E4%BB%94%E5%BA%97/WwxrEUEFVte5TKXH9eIfPQ)

### 七股海產街

- 注意：餐廳選擇要出發前再挑，避免現場找太久。

- 注意：先查當日營業與排隊，避免 B 版卡在午餐。

### 三井 Outlet 餐廳

- 適合方案：C 午餐、A/B 轉場備案。

### 孔廟 / 府中街咖啡甜點

- 類型：下午茶、飲料、散步。
- 適合方案：A 下午。
- 判斷：A 版下午需要一個坐下休息的空間，咖啡甜點最適合承接。
- 注意：可現場看，也可出發前先挑 1-2 家。

## 戶外與附近備案

### 延平郡王祠

- 類型：市區戶外主錨點、歷史建築、御守。
- 適合方案：A。
- 建議停留：45-60 分鐘。
- 判斷：和既有 A 版吃喝動線最順。適合午餐後第一站，接孔廟或府中街。
- 注意：6 月午後熱，不建議停留過久。台南旅遊網列 08:30-17:30、免門票。
- 來源：[台南旅遊網：延平郡王祠](https://www.twtainan.net/zh-tw/attractions/detail/792/)

### 孔廟文化園區 / 府中街

- 類型：市區散步、古蹟、甜點飲料、街區。
- 適合方案：A。
- 建議停留：50-80 分鐘。
- 判斷：最適合接在延平郡王祠後面，台南感強，也容易插入甜點、飲料、咖啡休息。
- 注意：戶外樹蔭有限，若太熱可縮短，轉林百貨或台南市美術館。

### 大南門 / 碑林 / 臺南放送局

- 類型：歷史散步、短戶外。
- 適合方案：A。
- 建議停留：30-50 分鐘。
- 判斷：想在郡王祠與孔廟之外再加一個戶外點時，這組比跑遠更合理。
- 注意：只挑一段即可，不要把下午塞成古蹟馬拉松。

### 創意積木夢工場｜LEGO 樂高

- 類型：近郡王祠室內備案。
- 適合方案：A 抗熱/雨備案。
- 地址：臺南市中西區樹林街二段 268 號。
- 判斷：這個點不是純戶外，但和延平郡王祠很近，很適合當天氣不舒服時的附近緩衝。
- 注意：營業狀態、是否需要預約或低消，出發前再看地圖資訊或店家公告。
- 來源：[地圖參考](https://maps.app.goo.gl/Eea9MqTG9Wi6UY3W6)

### 河樂廣場 / 海安路

- 類型：晚間戶外、散步拍照。
- 適合方案：A 晚餐後、宵夜前。
- 建議停留：30-60 分鐘。
- 判斷：晚上比下午舒服，若晚餐在中西區，接河樂廣場或海安路散步很自然。
- 注意：白天不優先，水景與現場狀態以當日公告為準。

### 藍晒圖文創園區

- 類型：拍照、文創小店、晚間備案。
- 適合方案：A 傍晚或晚餐後。
- 建議停留：30-45 分鐘。
- 判斷：想要一個好拍的城市點，可以放傍晚或晚上，不建議大太陽時硬排。
- 注意：園區店家營業時間各自不同。

### 巴克禮紀念公園

- 類型：公園、樹蔭、湖景。
- 適合方案：C、高鐵周邊備案。
- 建議停留：40-70 分鐘。
- 判斷：若 C 版想補戶外，巴克禮比市區街區更有綠地感，也相對有樹蔭。
- 注意：仍要看當天天氣與交通。
- 來源：[巴克禮公園官方網站](https://tainan-barclaypark.squarespace.com/)

### 安平樹屋 / 安平古堡 / 漁光島

- 類型：安平戶外支線、歷史街區、夕陽。
- 適合方案：A 改安平半日、另一天、或自駕時。
- 建議停留：2-4 小時含交通。
- 判斷：戶外感更強，但會吃掉市區吃喝時間。若大家想去安平，就應該整段調整，不要硬塞。
- 注意：漁光島適合傍晚，回程與晚餐要一起重排。

### 四草綠色隧道

- 類型：生態船、紅樹林、半日戶外。
- 適合方案：B 海線或另一天半日遊。
- 建議停留：2-3 小時含交通。
- 判斷：記憶點很強，但船班、排隊、交通都要抓緩衝，不適合 A 版市區午後硬塞。
- 注意：下雨、風況與船班會影響體驗。

## 景點候選

### 孔廟商圈 / 府中街

- 類型：市區散步。
- 適合方案：A 下午。
- 判斷：可以接延平郡王祠，不需要專程繞路。

### 奇美博物館

- 類型：室內景點。
- 適合方案：C 下午、雨天備案。
- 判斷：高鐵周邊版很適合，不想曬太陽時尤其好用。
- 注意：需要看展覽興趣、票務與休館資訊。
- 來源：[奇美博物館參觀指南](https://www.chimeimuseum.org/visit)

### 十鼓仁糖文創園區

- 類型：園區、表演、拍照。
- 適合方案：C 下午。
- 判斷：比奇美更動態，適合想拍照和走園區的人。
- 注意：門票、演出時間與營業時間需確認。
- 來源：[十鼓仁糖文創園區參考](https://tainan.lakeshore.com.tw/local-tour/activities/tendrum/)

### 井仔腳瓦盤鹽田

- 類型：海線景點。
- 適合方案：B 下午。
- 判斷：可接七股鹽山，但只建議天氣好、自駕時安排。
- 注意：不要和太多海線點硬塞。

### 青鯤鯓扇形鹽田

- 類型：海線拍照備案。
- 適合方案：B 下午。
- 判斷：若想把海線感拉滿，可和井仔腳二選一。

### 神農街 / 國華街

- 類型：市區備案。
- 適合方案：A 晚餐前後。
- 判斷：時間多再去，不必當主軸。
