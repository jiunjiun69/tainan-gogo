# 6/20 台南行程企劃簡報

最後更新：2026-06-13

## 目前狀態

- 旅行日期是 2026-06-20 週六，位於 6/19-6/21 端午三天連假中間。
- 主方向偏向台南，高雄比較像備用想法。
- 有人會搭高鐵，所以行程最好顧到高鐵動線。
- 6/20 位於連假中間，餐廳可能需要先訂位。
- 目前更像 A/C 二選一，A 市區穩健版機率較高，C 高鐵周邊版保留省心備案。
- 餐飲討論已收斂到「吃完早餐後出發、午餐與晚餐先定、宵夜可有可無」。
- 新增候選包含法國鬍子、牛肉湯、鱔魚意麵、國華街小吃、甜點飲料、夜市小吃。
- 戶外主點以延平郡王祠為錨點，並整理郡王祠附近可切換的室內備案。
- 「創意積木夢工場｜LEGO 樂高」位於臺南市中西區樹林街二段 268 號。它比較像抗熱/雨天備案，不是純戶外主點。

## 我的判斷

最適合先推 **A. 市區穩健版**，並把 **C. 高鐵周邊版**保留成省心備案。

市區版不用先解決開車、炎熱海線、回高鐵時間等大問題，阻力最低。它也最能承接目前想做的事情：午餐或早午餐、延平郡王祠御守、國華街/孔廟散步、晚餐海鮮，以及可有可無的宵夜。

七股鹽山不是不好，甚至因為 6/20 剛好落在七股鹽山官方「鹹轉海風祭」活動期間，變得比一般日子更值得考慮。但它需要車、耐熱、時間共識，所以比較適合當「大家想真的出去跑點」時才啟動的 B 方案。

## 戶外判斷

6 月台南午後不適合硬排長時間戶外，戶外應該做成「短段、可中斷、旁邊有飲料或室內點」的結構。

推薦戶外主線：

1. 延平郡王祠：A 版下午錨點。
2. 孔廟文化園區 / 府中街：距離近，台南感強，可接甜點飲料。
3. 大南門 / 碑林 / 臺南放送局：想多一點歷史散步時加，但不要硬塞太久。
4. 河樂廣場 / 海安路：更適合晚餐後或傍晚，不建議當下午曝曬主點。

附近備案：

- 創意積木夢工場｜LEGO 樂高：近郡王祠室內備案，適合太熱或下雨時補一段。
- 林百貨、台南市美術館：戶外撐不住時的冷氣備案。
- 藍晒圖文創園區：適合傍晚或晚上拍照。

遠距離備案：

- 安平樹屋、安平古堡、漁光島、四草綠色隧道：都值得去，但比較像半日支線，不適合在 A 版午後硬塞。
- 巴克禮紀念公園：更適合 C 版或高鐵周邊備案。

## 頁面定位

這個專案的首頁應該是可以直接打開看的行程參考頁，不是任務看板。

首頁重點：

- 先看 A/C 兩個最可能方向，B 海線保留備案。
- 每個方向都有完整時程草案。
- 吃喝和景點分開成候選池。
- 戶外點位有推薦排序、午後時間線、熱/雨備案。
- 最後只列出發前必須確認的事項。

## 必須確認

- 實際參加人數。
- 大家抵達高鐵台南站或台南市區的時間。
- 是否有人開車，或全程靠台鐵、計程車、YouBike。
- 吃完早餐後的出發時間與集合時間。
- 午餐要法國鬍子、牛肉湯、鱔魚意麵，還是國華街小吃。
- 晚餐海鮮要生魚片、魚湯、熱炒，或避開蝦蟹類。
- 宵夜要牛肉湯、夜市、打茶趣，還是直接回程。
- 戶外主線是否採用「延平郡王祠 → 孔廟/府中街 → 大南門或河樂廣場」。
- 太熱或下雨時，大家是否接受 LEGO、林百貨、美術館備案。
