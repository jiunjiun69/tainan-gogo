# 6/20 台南行程企劃簡報

最後更新：2026-06-05

## 目前狀態

- 旅行日期是 2026-06-20 週六，位於 6/19-6/21 端午三天連假中間。
- 主方向偏向台南，高雄比較像備用想法。
- 有人會搭高鐵，所以行程最好顧到高鐵動線。
- 6/20 位於連假中間，餐廳可能需要先訂位。
- 既有候選包含生魚片、打茶趣烤奶、七股鹽山、延平郡王祠御守。

## 我的判斷

最適合先推 **A. 市區穩健版**。

市區版不用先解決開車、炎熱海線、回高鐵時間等大問題，阻力最低。延平郡王祠適合安排在市區動線中；打茶趣則離高鐵台南站近，適合當晚茶或最後一站。

七股鹽山不是不好，甚至因為 6/20 剛好落在七股鹽山官方「鹹轉海風祭」活動期間，變得比一般日子更值得考慮。但它需要車、耐熱、時間共識，所以比較適合當「大家想真的出去跑點」時才啟動的 B 方案。

這個專案的首頁應該是可以直接打開看的行程參考頁，不是任務看板。

- 每個方向都有完整時程草案。
- 吃喝和景點分開成候選池。
- 最後只列出發前必須確認的事項。

## 必須確認

- 實際參加人數。
- 大家抵達高鐵台南站或台南市區的時間。
- 是否有人開車，或全程靠台鐵、計程車、YouBike。
- 主餐預算範圍。
- 是否一定要吃生魚片/海鮮。
- 打茶趣要當晚餐、晚茶，還是只是備案。
