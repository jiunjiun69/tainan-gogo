# A/B/C 行程選項

最後更新：2026-06-13

## A. 市區穩健版

**目前最可能。** 適合大多數人，不需要先解決開車問題，也比較不怕天氣臨時變差。餐飲可以完整接成午餐、下午茶、晚餐海鮮與彈性宵夜。

| 時間 | 安排 | 備註 |
| --- | --- | --- |
| 早上 | 吃完早餐後出發 | 若連假人潮較多，可以提早到市區會合。 |
| 11:30-13:20 | 午餐/早午餐：法國鬍子、牛肉湯、鱔魚意麵、國華街小吃 | 想坐著聊就選早午餐；想吃台南味就走小吃。 |
| 14:00-15:00 | 延平郡王祠 | 可看御守，市區低負擔景點。不要停太久，避免午後太熱。 |
| 15:05-16:25 | 孔廟、府中街、大南門三選二 | 這段是戶外主線核心；中間可以插甜點、飲料或咖啡休息。 |
| 16:25-17:20 | 創意積木夢工場｜LEGO 樂高、林百貨、台南市美術館，或繼續府中街 | 視天氣切換。LEGO 靠近郡王祠，適合抗熱/雨備案。 |
| 17:40-19:20 | 晚餐：海鮮、魚湯、熱炒或生魚片 | 建議先訂晚餐，蝦蟹類不要當唯一主菜。 |
| 20:00 後 | 宵夜：牛肉湯、花園/武聖夜市、打茶趣晚茶，或河樂廣場/海安路散步 | 可有可無，依體力和回程時間決定。 |

適合：

- 大家想好好吃飯、走走、放鬆。
- 不想跑太遠，或不想把車程拉太長。
- 想保留最多臨場調整空間。
- 想把御守、市區小吃、海鮮晚餐都放進同一天。
- 想要有戶外，但不想把 6 月午後排成長時間曝曬。

風險：

- 市區餐廳連假可能排隊，需要先訂主餐。
- 如果午餐、下午茶、晚餐、宵夜都想吃，份量要克制。
- 戶外若太熱，要果斷切室內備案，不要硬走完整段。

## B. 七股海線版

適合想把「去台南玩」做得更像出遊的一天，前提是以自駕為主，並且大家都能接受較長車程和 6 月日曬。

| 時間 | 安排 | 備註 |
| --- | --- | --- |
| 10:30-11:20 | 自駕集合出發 | 先確認停車、車程和乘車人數。 |
| 11:30-12:50 | 午餐：七股海產街、阿月虱目魚粥、黑琵食堂方向 | 海線餐廳要先看營業和排隊狀況。 |
| 13:10-14:40 | 七股鹽山 | 6/20 落在官方活動期間，適合當主景點。 |
| 15:00-16:30 | 井仔腳瓦盤鹽田、青鯤鯓扇形鹽田或四草綠色隧道 | 天氣太熱就縮短或跳過；四草需注意船班與排隊。 |
| 17:30-19:00 | 回市區或高鐵周邊晚餐 | 這段要抓車程與塞車緩衝。 |
| 20:00 後 | 高鐵回程，或有精神再接打茶趣 | 不要把最後一站排太硬。 |

適合：

- 自駕。
- 想拍照、跑景點、有出遊感。
- 大家能接受 6 月海線日曬。
- 想把戶外當主軸，而不是市區吃喝當主軸。

風險：

- 交通成本最高。
- 天氣不佳時體驗會掉很多。
- 吃飯和景點都要留緩衝，不適合塞滿。

## C. 高鐵周邊版

第二候選，適合有人晚到、早走，或大家比較想少移動，不想進市區排隊。

| 時間 | 安排 | 備註 |
| --- | --- | --- |
| 早上 | 吃完早餐後出發 | 不進市區時，集合壓力最低。 |
| 12:00-13:20 | 高鐵台南站集合，三井 Outlet 午餐 | 集合最容易，餐廳選擇多。 |
| 13:30-16:00 | 奇美博物館或十鼓仁糖文創園區 | 奇美偏室內穩，十鼓偏動態好拍。 |
| 16:30-17:10 | 回高鐵/三井周邊休息，或加巴克禮紀念公園短散步 | 補飲料、甜點或買東西。若想補戶外，可選樹蔭較多的公園段。 |
| 17:30-19:30 | 打茶趣 | 烤奶與圍爐煮茶適合當晚餐或晚茶。 |
| 20:00 後 | 回高鐵台南站，或市區宵夜備案 | 想吃台南味再加碼牛肉湯或夜市。 |

適合：

- 搭高鐵的人多。
- 大家想少移動。
- 想把打茶趣放進主軸。
- 想要輕量戶外，但不想進台南市區。

風險：

- 台南市區感比較少。
- 如果大家其實想吃傳統台南小吃，C 版會比較不滿足。
- 巴克禮公園和奇美外圍仍要看當天天氣，不要硬排。
