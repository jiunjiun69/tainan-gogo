# A/B/C 行程選項

最後更新：2026-06-06

## A. 市區穩健版

**目前最可能。** 適合大多數人，不需要先解決開車問題，也比較不怕天氣臨時變差。餐飲可以完整接成午餐、下午茶、晚餐海鮮與彈性宵夜。

| 時間 | 安排 | 備註 |
| --- | --- | --- |
| 早上 | 早餐各自處理 | 若連假人潮較多，可以提早出門再會合。 |
| 11:30-13:20 | 午餐/早午餐：法國鬍子、牛肉湯、鱔魚意麵、國華街小吃 | 想坐著聊就選早午餐；想吃台南味就走小吃。 |
| 14:00-15:10 | 延平郡王祠 | 可看御守，市區低負擔景點。 |
| 15:20-17:00 | 孔廟、府中街、國華街甜點飲料 | 豆花、冰淇淋、紅茶、黑糖奶都可當休息點。 |
| 17:40-19:20 | 晚餐：海鮮、魚湯、熱炒或生魚片 | 建議先訂晚餐，蝦蟹類不要當唯一主菜。 |
| 20:00 後 | 宵夜：牛肉湯、花園/武聖夜市，或打茶趣晚茶 | 可有可無，依體力和回程時間決定。 |

適合：

- 大家想好好吃飯、走走、放鬆。
- 沒有車或不想跑太遠。
- 想保留最多臨場調整空間。
- 想把御守、市區小吃、海鮮晚餐都放進同一天。

風險：

- 市區餐廳連假可能排隊，需要先訂主餐。
- 如果午餐、下午茶、晚餐、宵夜都想吃，份量要克制。

## B. 七股海線版

適合想把「去台南玩」做得更像出遊的一天，但前提是有人願意開車或大家接受包車/叫車。

| 時間 | 安排 | 備註 |
| --- | --- | --- |
| 10:30-11:20 | 集合、取車或叫車 | 沒有交通共識就不要選 B。 |
| 11:30-12:50 | 午餐：七股海產街、阿月虱目魚粥、黑琵食堂方向 | 海線餐廳要先看營業和排隊狀況。 |
| 13:10-14:40 | 七股鹽山 | 6/20 落在官方活動期間，適合當主景點。 |
| 15:00-16:30 | 井仔腳瓦盤鹽田或青鯤鯓扇形鹽田 | 天氣太熱就縮短或跳過。 |
| 17:30-19:00 | 回市區或高鐵周邊晚餐 | 這段要抓車程與塞車緩衝。 |
| 20:00 後 | 高鐵回程，或有精神再接打茶趣 | 不要把最後一站排太硬。 |

適合：

- 有車。
- 想拍照、跑景點、有出遊感。
- 大家能接受 6 月海線日曬。

風險：

- 交通成本最高。
- 天氣不佳時體驗會掉很多。
- 吃飯和景點都要留緩衝，不適合塞滿。

## C. 高鐵周邊版

第二候選，適合有人晚到、早走，或大家比較想少移動，不想進市區排隊。

| 時間 | 安排 | 備註 |
| --- | --- | --- |
| 早上 | 早餐各自處理 | 不進市區時，集合壓力最低。 |
| 12:00-13:20 | 高鐵台南站集合，三井 Outlet 午餐 | 集合最容易，餐廳選擇多。 |
| 13:30-16:00 | 奇美博物館或十鼓仁糖文創園區 | 奇美偏室內穩，十鼓偏動態好拍。 |
| 16:30-17:10 | 回高鐵/三井周邊休息 | 補飲料、甜點或買東西。 |
| 17:30-19:30 | 打茶趣 | 烤奶與圍爐煮茶適合當晚餐或晚茶。 |
| 20:00 後 | 回高鐵台南站，或市區宵夜備案 | 想吃台南味再加碼牛肉湯或夜市。 |

適合：

- 搭高鐵的人多。
- 大家想少移動。
- 想把打茶趣放進主軸。

風險：

- 台南市區感比較少。
- 如果大家其實想吃傳統台南小吃，C 版會比較不滿足。
